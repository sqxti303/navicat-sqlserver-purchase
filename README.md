## 三个版本怎么选

很多人买的时候搞不清楚标准版和企业版有什么区别，买了才发现缺功能，或者多花了钱买了用不到的东西。这里直接帮你理清楚。

Navicat for SQL Server 分三个版本：**标准版（Standard）、企业版（Enterprise）、非商业版（Non-Commercial）**。版本之间的核心差异只有两点：

- **数据建模工具**：仅企业版和非商业版有
- **商业智能工具 BI**：仅企业版和非商业版有

标准版其实已经涵盖了日常数据库管理的所有核心功能，不需要画模型、不需要做 BI 报表的话，标准版够用了。

非商业版功能上和企业版一样，但**仅限教育机构和非营利组织使用**，购买时可能需要提交资质证明，普通商业用途不能买。
